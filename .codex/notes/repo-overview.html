project-degen-bot Repo Overview

This fork starts from BowTiedDevil/degenbot, a Python toolkit for EVM pool modeling, state tracking, and arbitrage calculations. It is useful for building a Curve arbitrage research engine, but it is not yet a finished flash-loan bot or VPS daemon.

Current Purpose

  • Model DEX pool state and swap math across Uniswap, Curve, Aerodrome, PancakeSwap, SushiSwap, Balancer, Camelot, SwapBased, and related EVM protocols.
  • Calculate whether a route is profitable before execution.
  • Support local fork testing, database-backed pool metadata, and CLI workflows.
  • Provide a base for building a future Curve Finance flash-loan arbitrage system.

Main Folders

  • src/degenbot/: main Python package.
  • rust/: maturin-built Rust extension.
  • docs/: config, CLI, Aave, and arbitrage notes.
  • tests/: protocol and integration tests.
  • contract_reference/: protocol contract references.
  • .codex/: local documentation for this fork.

Key Modules

  • connection/: Web3/RPC access.
  • config.py: ~/.config/degenbot/config.toml settings.
  • curve/: Curve V1 StableSwap pool modeling.
  • arbitrage/: route profit calculations.
  • aave/: Aave state, math, events, and processors.
  • database/: SQLAlchemy models and migrations.

Curve Arbitrage Flow

  • Create Web3/RPC configuration for the target chain.
  • Instantiate relevant Uniswap and Curve pool objects from on-chain addresses.
  • Use CurveStableswapPool to model Curve balances, fees, tokens, and output estimates.
  • Use UniswapCurveCycle for paths shaped like Uniswap -> Curve -> Uniswap.
  • Call calculate() to get optimal input size, estimated profit, state block, and swap amounts.
  • Use generated payloads only after validating them against an executor contract and mainnet-fork simulation.

Still Needed For A Live VPS Bot

  • Route scanner and strategy runner.
  • Flash-loan executor contract and contract tests.
  • Gas, slippage, flash-loan premium, and failed-transaction cost checks.
  • Private relay or protected transaction submission.
  • Secret management for wallet keys and RPC credentials.
  • systemd or container deployment, logs, alerts, and a kill switch.

Useful Commands

  • uv sync: install/sync dependencies.
  • uv run degenbot --help: inspect the CLI.
  • just test-python: run Python tests.
  • just test-rust: run Rust tests.
  • just lint: run clippy, ruff, and mypy.
